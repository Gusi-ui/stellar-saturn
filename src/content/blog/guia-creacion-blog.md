¡Bienvenido al sistema de blog de **DiverMataró**! Esta guía te enseñará cómo crear artículos ricos y atractivos para nuestra comunidad.

## 1. Estructura Básica (El Frontmatter)

Todo artículo comienza con una sección especial entre tres guiones (`---`) llamada **Frontmatter**. Aquí definimos los datos clave del artículo:

```yaml
---
title: 'Mi Título Increíble'
description: 'Un resumen corto de una o dos frases.'
publishDate: 2024-03-21
author: 'Tu Nombre'
category: 'Noticias'
tags: ['etiqueta1', 'etiqueta2']
image: '/images/blog/mi-foto.jpg'
altText: 'Descripción de la imagen'
---
```

> **Nota:** La categoría debe ser una de las siguientes: `Actividades`, `Eventos`, `Reivindicaciones`, `Noticias` o `Recursos`.

## 2. Formateando el Texto

Escribir en **Markdown** es muy sencillo. Aquí tienes los elementos más comunes:

### Negritas y Cursivas

Para resaltar texto usa **asteriscos dobles** para negrita y _asteriscos simples_ para cursiva.

### Listas

Puedes crear listas desordenadas con guiones:

- Elemento uno
- Elemento dos
  - Sub-elemento

O listas ordenadas con números:

1. Primer paso
2. Segundo paso
3. Tercer paso

### Enlaces

Para crear un enlace, usa esta estructura: [Texto del enlace](https://destino.com).

## 4. Citas y Destacados

Si quieres resaltar una frase importante o una cita, usa el símbolo mayor que (`>`):

> "La inclusión no es dejar entrar, es dar la bienvenida."

## 5. Código (Opcional)

Si necesitas mostrar código técnico, usa tres comillas invertidas:

```javascript
console.log('¡Hola, mundo inclusivo!');
```
